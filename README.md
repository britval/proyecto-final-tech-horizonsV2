# Proyecto Final - Tech Horizons
## Análisis de Datos para Impacto Social usando los ODS
Créditos: Ing. Carlos Campbell - Coordinador de actividades académicas


### Descripción General
Este repositorio contiene todos los recursos necesarios para desarrollar tu **Proyecto Final** del curso de Ciencia de Datos y Analítica con Python. El objetivo es aplicar todas las habilidades aprendidas para crear una solución que genere **impacto social** alineada con los **Objetivos de Desarrollo Sostenible (ODS)**.

### Objetivo del Proyecto
Desarrollar una **solución basada en datos** que aborde una problemática real relacionada con al menos uno de los 17 ODS de la ONU, demostrando competencias técnicas y conciencia social.

## Características del Proyecto

### Requisitos Obligatorios
- **Trabajo en equipo:** 6-7 integrantes
- **Enfoque en ODS:** Alineado con uno o más de los 17 Objetivos de Desarrollo Sostenible
- **Análisis de datos:** Uso de pandas, matplotlib/seaborn para explorar y visualizar datos
- **Solución técnica:** Dashboard, aplicación web, o notebook interactivo
- **Impacto medible:** Propuesta clara de cómo la solución genera impacto
- **Documentación:** Código documentado y explicación clara del proyecto

### Elementos Creativos (Opcionales)
- **Dashboard interactivo** con Streamlit, Dash, o similar
- **Modelos predictivos** simples con scikit-learn
- **Visualizaciones web** con Plotly o D3.js
- **Responsividad** para dispositivos móviles
- **Elementos multimedia** (videos, audio, animaciones)
